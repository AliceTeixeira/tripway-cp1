# TripWay

Projeto desenvolvido para o Checkpoint 1 da disciplina de Front-end.

## Sobre

Site multipágina de uma agência de viagens fictícia chamada TripWay. O usuário pode ver destinos, conhecer os pacotes oferecidos e preencher um formulário para solicitar uma reserva.

## Estrutura

```
tripway/
├── index.html
├── pages/
│   ├── destinos.html
│   ├── pacotes.html
│   └── contato.html
├── images/
│   ├── banner.jpg
│   ├── praia.jpg
│   ├── montanha.jpg
│   ├── cidade.jpg
│   ├── pacotes.jpg
│   └── contato.jpg
└── README.md
```

## Páginas

- **Home** - apresentação da empresa
- **Destinos** - 3 destinos com nome, descrição e imagem
- **Pacotes** - lista de serviços oferecidos
- **Contato** - formulário de reserva
